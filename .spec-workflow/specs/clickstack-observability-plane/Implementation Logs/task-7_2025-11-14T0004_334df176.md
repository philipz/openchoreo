### Integrations

#### Integration
- **Description:** Backstage plugins integrate with HyperDX dashboards via signed URLs from Observer API. URLs are fetched from /api/hyperdx/link endpoint, embedded in iframes with proper sandbox attributes, and refreshed before 15min expiration. Supports CORS configuration for secure cross-origin embedding.
- **Frontend Component:** Backstage Plugins (TypeScript/React)
- **Backend Endpoint:** Observer API /api/hyperdx/link
- **Data Flow:** Backstage component mounts → Fetch signed URL from Observer API → Embed in iframe with sandbox → Auto-refresh every 10min → HyperDX renders ClickStack dashboard

#### Integration
- **Description:** CLI (choreoctl) maintains full backward compatibility by querying Kubernetes pods directly for real-time logs, not Observer API. This ensures fastest access. For historical search, users are directed to Observer API or HyperDX dashboard. No CLI code changes required.

- **Backend Endpoint:** Kubernetes Pod Log API (not Observer API)
